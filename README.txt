HOW TO EXECUTE CODE
All files should be deployed to the same directory. Go to the correctdirectory in your terminal and install the required node modules bytyping "npm install". Start the server by typing "npm start" and thenvisit http://localhost:3000/ in a browser.The website hasbeen tested using Chrome and no errors are known to exist with otherbrowsers.
